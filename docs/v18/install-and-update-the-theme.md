
### Install & Update the Theme


Prior to installing Theme Clarico Vega, the latest versions of Odoo and Theme Clarico Vega, together with all of its submodules, were downloaded.


**1. Install Theme**


{:.alert-warning} 
> 
> #### NOTE
> 
> **Technical Steps to Install/Update Theme Clarico Vega:**
> 
> 
> 1. 1.Log in with your account and then you can download the purchased app order details. Go to give link https://www.odoo.com/my/orders
> 2. Copy and paste theme\_clarico\_vega and emipro\_theme\_base code into your custom module (Add/Replace).  
> 
> 3. Restart the odoo services. (Ignore if you are using Odoo.sh)  
> 
> 4. Go to Apps -> Update List (Please start debug mode first otherwise it will not show this menu).  
> 
> 5. Find emipro\_theme\_base module & Install/Upgrade that module.  
> 
> 6. Install/Update a theme Clarico Vega from Website -> Editor -> Edit -> Theme tabs -> Website section -> Click Switch theme and select theme clarico vega.
> 
> 
>

**2. Update The Clarico Vega** 


Download the latest version from Odoo Store and replace it with your existing Theme source code, follow the above steps.
